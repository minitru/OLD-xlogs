00 What files do I need to upload to my server?

Please upload the services-plugin folder to your server. In this folder you will find the following subfolders containing all of the items content:

  • js
  • css
  • assets

You could use your own jQuery but we recommend loading it directly from the Google ressources (see later in this documentation).

 

01 Implement the jQuery:

Add the following lines to your HTML <HEAD>:

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7/jquery.js"></script>

 

02 Add services js and css files to your HTML page:

Also in the <HEAD> section:

<!-- ANIMATE AND EASING LIBRARIES -->
<script type="text/javascript" src="services-plugin/js/jquery.easing.1.3.js"></script>
<script type="text/javascript" src="services-plugin/js/jquery.cssAnimate.mini.js"></script>

<!-- TOUCH AND MOUSE WHEEL SETTINGS -->
<script type="text/javascript" src="services-plugin/js/jquery.touchwipe.min.js"></script>
<script type="text/javascript" src="services-plugin/js/jquery.mousewheel.min.js"></script>

<!-- jQuery SERVICES Slider -->
<script type="text/javascript" src="services-plugin/js/jquery.themepunch.services.js"></script>
<link rel="stylesheet" type="text/css" href="services-plugin/css/settings.css" media="screen" />

This contains the JS and CSS for the plugin itself (jquery.themepunch.services.js,settings.css) and some helping modules (jquery.easing.1.3.js,jquery.cssAnimate.mini.js,jquery.mousewheel.js,jquery.touchwipe.min.js).

 

03 Create the surrounding DIV for your item :

Put it in the HTML:

<div id="services-example-1" class="theme1">...</div>

05 Call the jQuery Plugin to build the Slider:

Put this code underneath the above created DIV:

			<!--
			##############################
			 - ACTIVATE THE SERVICES HERE -
			##############################
			-->
			<script type="text/javascript">
				
				$(document).ready(function() {
					$.noConflict();					 									
					
				jQuery('#services-example-1').services(
				{										
							width:920,
							height:295,							
							slideAmount:5,
							slideSpacing:30,							
							touchenabled:"on",
							mouseWheel:"on",
							transition:1,
							slideshow:3000,
							callBack:function() {}
				});
			</script>
