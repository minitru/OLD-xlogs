01 The Default parameters of the Plugin

  • width/height - Sets the Size of the Plugin. Padding, shadows etc will added from this size.
  • slideAmount - How many slides visible at a time
  • slideSpacing - Padding between the slides
  • touchenabled - Enable the touch swipe detection on/off
  • mouseWheel - Enable mousewheel detection on/off
  • slideshow - Timer interval of automatic rotation in ms (0 = off, 2000 = 2 sec.)
  • callback - Enter your JS code here if needed (for example cufon.refresh())
<script type="text/javascript">
   $(document).ready(function() {
					$.noConflict();					 									
					
			jQuery('#services-example-1').services(
				{										
							width:920,
							height:295,							
							slideAmount:5,
							slideSpacing:30,							
							touchenabled:"on",
							mouseWheel:"on",
							slideshow:3000,
							callBack:function() {}
				});
						</script>

02 The Services Declaration

A <div> that is called via the above jQuery call declares the the service. This div wraps the unsorted list items (banner items). The theme added here in class defines the optics.

An Example:

<div id="services-example-1" class="theme1">
  <ul>
    <li ...>
      ...
    </li>
  </ul>
</div>


03 Slider items

<li> - Every list item represents a new slider item.

It is divided into two parts: A normal/preview view and the detailed view which opens when the "readmore" button is been clicked.

You can basically put in the HTML of your choice in both containers. Remember to use the preview/normal stuff directly in the <li>, the detailed view elements belong into the DIV with the class page-more.
