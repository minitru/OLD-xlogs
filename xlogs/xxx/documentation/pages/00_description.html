Showbiz Business Carousel jQuery Plugin

What are we talking about here?

This is a highly customizable plugin to show you or your customers services, portfolio items, blog contents... basically all business information thinkable.

So what is so new?

What makes this Carousel unique is that you can define everything via CSS or HTML or jQuery plugin option which makes it 100% customizable.

You have a small preview view that can be expanded to a rich detailed panoramic view. All fits nicely in the designated area.

Does this run on all browsers and on mobiles?

Sure it works on every modern browser (including IE7/8) and on iOS(iPad,iPhone) and Android mobile devices. Touch swipe for iOS and Android mobile devices is of course included.

What are the features?

  • Image fully resizable
  • Using CSS Animation with FallBack to jQuery
  • Unlimited Slides
  • iPhone & Android Swipe Touch enabled
  • jQuery Conflict free plugin
  • Customizable 100% via Plugin Parameters / CSS / HTML
  • Easy installation in your Website
  • Special Easing
  • Example Page included
